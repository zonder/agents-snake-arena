Issue #99 · lobby-clarity-polish

Lobby redesign concepts

These concepts all keep the current product’s retro-arcade direction, 4-letter room code flow, ready logic, reconnect messaging, and desktop/mobile support — but push much harder on pre-match clarity and lobby identity. The goal is to give the stakeholder real choices, not minor cosmetic tweaks.

Concept A — Mission Control
Best for clarity-first staging with explicit “what happens next” guidance.
Concept B — Arena Cabinet
Best for stronger retro-arcade drama with the match board preview as the centerpiece.
Concept C — Neon Marquee
Best for a bolder brand identity with a stronger logo/header zone and event-poster feel.
Concept A

Mission Control

A clean pre-match staging layout. The room code and next-step explanation dominate the first scan. Players sit in a simple two-card roster while a progress rail explains the flow: share code → second player joins → both ready → auto start.
